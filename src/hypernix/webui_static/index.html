Module Reference

PressureCookerV3

Advanced optimizer with quantization-aware training (QAT), gradient checkpointing, adaptive clipping, EMA shadowing, and distributed awareness.

StovetopV3CookerPlus

Enhanced stovetop variant optimized for stable convergence on consumer hardware.

Abbicus

Dynamic curriculum learning system that automatically adjusts sequence length and padding based on model size and dataset complexity.

ComputeFramework

Hardware abstraction layer supporting CUDA, MPS, CPU, and TPU backends with automatic DDP/ZeRO wrapping.

Tupperware

Dataset round-splitting utility for multi-phase training with per-round step budgets and learning rate schedules.

GGUF quantization pipeline supporting 30+ quantization types from F32 to IQ2_XXS.
